Easily add MLA style citations to your WordPress posts.

== Description ==
This plugin adds five shortcodes to your WordPress site. The first one generates a \"Works Cited\" section, three generate appropriately formatted citations for books, periodicals, and journals, and the last allows you to add notes to your citations.

This plugin also adds a shortcode (hly) for highlighting text in quotations, as well as shortcodes for clearing floats.

For complete usage instructions, please visit http://kjodle.net/wordpress/mla-citaitons/

== Installation ==
1. Upload the \'d12-mla-citations\' folder to \'/wp-content/plugins/\'
1. Activate the plugin through the \'Plugins\' menu in WordPress
1. That\'s it!
